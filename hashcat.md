# Hashcat

```
hashcat -m [HASH TYPE] -a [ATTACK MODE] [HASHFILE PATH] [WORDLIST PATH]
```

## Related Tools
Hashid  
```
hashid -m [HASH]
```
## Memo
*1 Hashidは、Hash-identifierや他のハッシュ解析ツールでは特定できないハッシュタイプも特定可能。また、解析結果にHashcatのモード番号を表示してくれる。
